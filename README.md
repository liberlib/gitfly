<p>
  <h1 align="center">Welcome to gitfly</h1>
  <p>gitfly is a tool used to automate your requests, with contributions from @hackwinable and @liberlib.
Its purpose is to automate your git requests in a single environment, while also enabling you to quickly perform actions such as commits, issues, and pull requests, and manage all requests in a GUI-based system. </p>
